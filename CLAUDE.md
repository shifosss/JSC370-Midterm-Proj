## Project Overview

JSC370 midterm project investigating **Option C — Clinical Trial Access Gap**: whether diabetes-related clinical trial sites are equitably distributed relative to disease burden across the United States. Uses programmatic data acquisition (APIs), EDA, and a reproducible Quarto pipeline. Options A, B, and D are **archived** (empty scaffolds remain but are not active).

## Active Option

**Option C** (`option-c-trial-access/`) — Clinical trial site density vs disease burden for diabetes, at both state and county levels.

**Primary APIs:** ClinicalTrials.gov v2, CDC PLACES (Socrata/SODA), CDC BRFSS (Socrata), US Census ACS 5-Year 2022, US Census 2020 Decennial, NPI Registry

## Key Datasets

### State-Level (`discover/modified_data/state_modeling_final.csv`)
- **51 rows** (50 states + DC) x **57 columns**
- Primary outcome: `trials_per_100k` (diabetes trial density per capita)
- Primary burden variable: `places_diabetes` (CDC PLACES age-adjusted prevalence)
- Equity metric: `coverage_residual` (OLS residual: observed minus expected trial density given burden)
- Equity metric: `mismatch_index` (|burden_rank - trial_rank|, both normalized to [0,1])
- SES covariates from ACS 2022: income, poverty, race/ethnicity, insurance, education, vehicle/internet access
- Infrastructure: `endo_per_100k`, `amc_count`, `has_ctsa`
- Trial breakdowns by phase, sponsor type, recruitment status

### County-Level (`discover/modified_data/county_modeling_final.csv`)
- **3,221 rows** x **42 columns**
- Primary access variable: `dist_nearest_trial_km` (Haversine distance to nearest trial site)
- 26.8% of counties have at least one diabetes trial site
- Same SES/health covariates as state level, plus `burden_decile`
- See `discover/County_level_data_documentation.md` for full column reference

## API Patterns

**ClinicalTrials.gov v2** — `https://clinicaltrials.gov/api/v2/studies`; use `query.cond` for condition, `pageSize` (max 1000), `pageToken` for cursor-based pagination. Check `/api/v2/version` for freshness.

**Socrata/SODA (CDC PLACES, CDC BRFSS)** — Page with `$limit` + `$offset`; always add `$order` for stable pagination. CDC endpoint pattern: `https://data.cdc.gov/resource/<dataset-uid>.json`.

**US Census ACS** — `https://api.census.gov/data/<year>/acs/acs5`; requires `key` param (free API key). Use `get` for variable list and `for`/`in` for geography filters.

**NPI Registry** — `https://npiregistry.cms.hhs.gov/api/`; filter by `taxonomy_description`; paginate with `skip` + `limit` (max 200). Endocrinology taxonomy: `207RE0101X`.

## Conventions

- Store API keys in `.env` (gitignored); load via `python-dotenv` or `Sys.getenv()` in R.
- Save raw API responses with descriptive filenames in `product/data/raw/`.
- Document pagination strategy and download date in the methods section of the QMD.
- Never imply causality from ecological (aggregate) associations.
- Keep scope controlled: diabetes condition area, state + county geographic levels, 2022-2025 time window.
- See `discover/State_level_data_documentation.md` and `discover/County_level_data_documentation.md` for full column-by-column references.
